## AT Command

AT+를 입력하고 COMMAND를 입력하고 ?를 붙이면 세팅된 정보를 가져오고 PARAMETER를 입력하면 원하는 값으로 세팅한다.

ex)AT+NAME? -> 세팅된 NAME을 보여줌

ex)AT+NAMEB01 -> NAME을 B01로 

AT+COMMAND

AT+NAME

AT+UUID

MAJOR 설정

AT+MARJ
